# OneWireUART\_Driver\_STM32

This is program for STM32 board to connect with Serial Servo Motors (Feetech STC/SCS) using Single Line UART communication mode

**Firmware** folder has the Code for STM32F103C8T6.

* Connect the STM32 with system using ST-Link
* Select the .elf file from ...\\OneWireUART\_Driver\_STM32\\Firmware\\Debug
* Download to the STM32f103c8t6

**Electronics** Folder has the Schematics regarding 1-Wire Uart circuit

* Schematic includes local components like very common tri state buffers based
* Also with 3.3V logic based components for redundant PCB under the name of International Components
